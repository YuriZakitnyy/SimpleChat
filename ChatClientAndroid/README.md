# SimpleChat Android Client

Android client for SimpleChat application using SignalR.

## Requirements
- Android Studio Arctic Fox or later
- Android SDK 24 or higher
- Java 8

## Setup
1. Open Android Studio
2. Select "Open an existing project"
3. Navigate to this directory
4. Wait for Gradle sync to complete
5. Build and run on emulator or device

## Configuration
Default server URL: http://10.0.2.2:5000/chathub (for emulator)
For physical device, use your computer's IP address.

## Features
- Connect to SignalR chat hub
- Send and receive text messages
- Message history
- Connection state management
- Settings persistence
